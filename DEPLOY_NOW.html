Add Custom Domain in Vercel

After deployment, add your custom domain:

  1. Go to Vercel Dashboard
  2. Click on your better-business-builder project
  3. Click Settings ‚Üí Domains
  4. Add domain: bbb.aios.is
  5. Click Add
‚ö†Ô∏è Vercel will show "Invalid Configuration" until you add DNS records (next step)

Configure DNS at Namecheap

Go to Namecheap and update DNS:

  1. Login to Namecheap.com
  2. Go to Domain List
  3. Click Manage next to aios.is
  4. Click Advanced DNS tab
  5. Click Add New Record

Add this CNAME record:

Type Host Value TTL
CNAME Record bbb cname.vercel-dns.com Automatic
‚úÖ Click Save All Changes at the bottom!
‚ÑπÔ∏è DNS changes can take 5-30 minutes to propagate. Usually it's very fast (2-5 minutes).
